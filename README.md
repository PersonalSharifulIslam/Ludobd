# LudoRex - Online Ludo Gaming Platform

A world-class online Ludo gaming platform with casino-style design, built with Next.js 14, Supabase, and deployed on Cloudflare Pages.

## Features

- 🎮 Real-time 2-player Ludo game
- 💰 Secure payment system (bKash, Nagad, Rocket)
- 👥 User authentication and profiles
- 🎁 Referral and bonus system
- 🎯 Admin panel with game control
- 🌟 Casino-style UI with animations
- 📱 Fully responsive design

## Tech Stack

- **Frontend**: Next.js 14 with React & TypeScript
- **Backend**: Supabase (PostgreSQL + Auth + Realtime)
- **Deployment**: Cloudflare Pages
- **Styling**: Tailwind CSS
- **Real-time**: Supabase Realtime WebSocket

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```
